# مواصفات شاشات المصادقة

## Login Page

### العناصر

- حقل identifier
- حقل password
- زر تسجيل الدخول
- رابط نسيت كلمة المرور
- رسائل أخطاء واضحة

### الحالات

- idle
- submitting
- invalid_credentials
- account_locked

## Forgot Password Page

### العناصر

- حقل identifier
- زر إرسال الرمز
- حقل OTP
- حقل كلمة المرور الجديدة
- حقل تأكيد كلمة المرور

### السلوك

- تنفيذ الخطوات على شاشة واحدة أو خطوتين حسب سهولة الاستخدام

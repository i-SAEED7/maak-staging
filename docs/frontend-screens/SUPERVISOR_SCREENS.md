# مواصفات شاشات المشرف

## Supervisor Dashboard

- بطاقات مؤشرات المدارس
- عدد الزيارات القادمة
- الخطط بانتظار الاعتماد
- الرسائل غير المقروءة

## Visits Page

- قائمة الزيارات
- فلتر التاريخ والحالة

- الانتقال إلى تفاصيل الزيارة

## IEP Plans Page

- قائمة الخطط بانتظار الاعتماد
- فلتر المدرسة والحالة
- إجراءات اعتماد أو رفض

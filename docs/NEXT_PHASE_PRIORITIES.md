# أولويات التنفيذ فور توفر الأدوات

## الأولوية 1: توليد المشاريع الأساسية

1. إنشاء Laravel 11 داخل `backend/`
2. إنشاء React + Vite + TypeScript داخل `frontend/`
3. تثبيت الحزم الأساسية فقط دون أي ميزات إضافية

## الأولوية 2: قاعدة البيانات والمصادقة

1. تحويل ملف SQL الحالي إلى `migrations`
2. إنشاء `roles`, `permissions`, `schools`, `users`
3. Seed للأدوار والصلاحيات المرجعية
4. تفعيل Sanctum
5. بناء تسجيل الدخول والخروج و`/auth/me`

## الأولوية 3: العزل متعدد المدارس

1. `SetTenant` middleware
2. `TenantContext` service
3. `BelongsToSchool` trait
4. أول Policies عملية على `School`, `User`, `Student`

## الأولوية 4: الواجهة الأولية

1. تفعيل Tailwind وRTL
2. بناء `AppShell`
3. بناء صفحات المصادقة
4. حماية المسارات حسب الدور

## الأولوية 5: أول وحدتين تشغيليتين

1. المدارس
2. المستخدمون

## تعريف إنجاز المرحلة التالية

تعتبر المرحلة التالية ناجحة عند اكتمال ما يلي:

- تسجيل الدخول يعمل
- الأدوار تعمل
- المدارس CRUD
- المستخدمون CRUD
- العزل يعمل حسب المدرسة
- واجهة عربية أولية قابلة للاستخدام

# قوالب جمع البيانات

هذه القوالب مخصصة لتجميع بيانات المدارس من الفرق التشغيلية أو من الإدارة قبل إدخالها للنظام.

## الصيغ المدعومة عمليًا

- `xlsx` للتعبئة اليدوية
- `csv` للتحويل السريع والاستيراد لاحقًا

## القوالب الحالية

- [schools_template.csv](./schools_template.csv)
- [users_template.csv](./users_template.csv)
- [students_template.csv](./students_template.csv)

## ملاحظات

- اجعل `school_code` ثابتًا بين الملفات كلها.
- استخدم `UTF-8` عند حفظ ملفات CSV.
- لا تغيّر أسماء الأعمدة إلا إذا اتفقنا على ذلك.

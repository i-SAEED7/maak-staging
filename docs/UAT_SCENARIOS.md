# سيناريوهات قبول المستخدم UAT

## 1. المدير العام ينشئ مدرسة ومستخدمًا

1. تسجيل الدخول كمدير عام
2. فتح صفحة المدارس
3. إنشاء مدرسة
4. فتح صفحة المستخدمين
5. إنشاء معلم وربطه بالمدرسة
6. التأكد من ظهور المدرسة والمستخدم

## 2. المعلم ينشئ طالبًا وخطة

1. تسجيل الدخول كمعلم
2. فتح صفحة الطلاب
3. إنشاء طالب
4. فتح محرر IEP
5. إنشاء مسودة
6. إرسالها للمراجعة

## 3. مدير المدرسة يراجع الخطة

1. تسجيل الدخول كمدير المدرسة
2. فتح صفحة الخطط
3. عرض الخطة
4. اعتمادها أو رفضها مع تعليق

## 4. المشرف يعتمد الخطة

1. تسجيل الدخول كمشرف
2. فتح قائمة الخطط
3. اعتماد الخطة
4. التأكد من ظهورها كمعتمدة

## 5. ولي الأمر يرى التقرير

1. تسجيل الدخول كولي أمر
2. فتح صفحة الأبناء
3. فتح التقارير
4. مشاهدة الخطة أو التقرير المنشور

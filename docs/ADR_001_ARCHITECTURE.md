# ADR 001: اختيار معمارية النظام

## الحالة

معتمد

## القرار

اعتماد:

- React SPA للواجهة
- Laravel API للخلفية
- PostgreSQL كقاعدة بيانات
- Redis للكاش والجلسات والصفوف
- MinIO للملفات

## السبب

- ملاءمة عالية للـ RBAC والـ CRUD المركب
- فصل واضح بين الواجهة والخلفية
- قابلية توسع جيدة
- مناسبة لفرق تطوير متوسطة الحجم

## الأثر

- نحتاج إلى تعريف واضح لعقود API
- نحتاج إلى ضبط محكم لـ tenancy وpolicies
- الاختبارات ستتوزع بين backend وfrontend

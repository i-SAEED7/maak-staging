# التسلسل التنفيذي العملي للمرحلة 2

## عند توفر أدوات الواجهة

1. توليد React + Vite + TypeScript
2. إضافة Tailwind وRTL
3. بناء `AppShell`
4. بناء `authStore`
5. بناء `api client`
6. بناء صفحات:
   - Login
   - Forgot Password
   - Admin Dashboard

7. إضافة route guards
8. ربط الواجهة الفعلية بالـ API
